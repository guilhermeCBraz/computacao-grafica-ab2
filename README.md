# Projeto Futebol AB2 Computação Gráfica

Projeto futebol aplicando texturas e iluminação

## Como compilar e executar

Para executar o projeto necessário ter a biblioteca OpenGL e o compilador g++ instalados no seu computador. A seguir é necessário abrir um terminal e dentro da pasta baixada do github, executar o seguinte  comando:

**g++ futebol.cpp std_image.cpp -o futebol -lGL -lGLU -lglut**

Assim, basta executar o arquivo compilado usando esse comando  no terminal:

**./futebol**


## Funcionalidades

Ao executar o projeto, um menu com os comandos será exibido na tela, mostrando as interações em cada um. Em todos eles é possível mover a bola com as teclas **WASD**, marcar um gol, mudar entre câmeras predefinidas com o **1** e **2** do teclado, ou move-las livremente com as setas, reiniciar o placar com **C** e ocultar o menu com essas instruções com a tecla **H**.
